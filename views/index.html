Características

  • Validación de información de tickets e identificación
  • Cambio de estado de abordaje de pasajeros
  • Finalización del proceso de abordaje con informes en PDF
  • Interfaz intuitiva para el personal de la aerolínea
  • Integración con sistemas de control de acceso

Sistema Dental de Citas y Expedientes

Desarrollé una aplicación web completa para una clínica dental con roles de cliente y administrador. El sistema gestiona citas, registros médicos y comunicaciones con los pacientes.

Características

  • Reserva de citas con notificaciones por correo electrónico
  • Autenticación de usuarios y actualizaciones de perfil
  • Registros de pacientes con carga de imágenes y seguimiento de diagnósticos
  • Interfaz administrativa para gestión completa de usuarios y citas
  • Sincronización en tiempo real de la disponibilidad de horarios

Tecnologías

Node.js OracleDB Docker Bootstrap JavaScript DigitalOcean Oracle Cloud

Resultados

El sistema permitió un proceso de programación de citas más rápido, redujo los errores de reserva y mejoró la accesibilidad a los registros de los pacientes.
